# Object margin / padding 을  % 와 em 단위 적용할 경우 

## % (퍼센티지) 값을 사용할 경우

% 값들은 부모 엘리먼트의 너비를 기준으로 한 비율 값을 계산이 되기때문에, 마진 속성을 가지지 않고 있고, 때문에 마진 값도 가질 수 없다.
기본적으로 margin 속성을 가지고 있지 않기 때문에 margin 값도 가질 수 없다.
